#Collection of useful shortcuts for vim

##Editing

* `o` - new line and insert
* `O` - new line and insert above

* `yy` - yank line
* `p` - paste after cursor
* `P` - paste before cursor

* `C-[` - exit mode

* `u` - undo
* `C-r` - redo

##Buffers/Windows

* `C-w =` - balance windows

##GREP

* `:grep` or `:vimgrep`
* `:cw` or `:copen`

* `:lgrep` `:lvimgrep`
* `:lw` or `:lopen`
