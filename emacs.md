# Emacs

  start a clojure repl
    lein repl

  emacs cheat sheet
  Files
    c-f     # open
    c-x c-s # save
    c-x c-c # close
    c-g     # god exists, and he is escape the command you messed up

  Cursor Movement
    m-b   # word backwards
    m-f   # word forwards
    c-x n # next line
    c-x p # prev line
    c-x f # move right
    c-x b # move left
    c-spc # mark selection start or unmark
    c-w   # kill selection

  Undo
    c-x u   # undo tree responds to up/down/q
    m-y #paste, responds to <cr>
    C-g C-_ # redo?
  Windows/Buffers
    c-x 1     # unsplit windows
    c-x 0     # close window
    c-x 3     # veritcal split
    c-x c-b   # list buffers
    c-x o     # switch windows

  Eval
    c-x c-e   # eval ) backwards
    m-x nrepl # connect to repl
    m-x cider # repl clojure

    etc
    c-m-f  # matching paren
